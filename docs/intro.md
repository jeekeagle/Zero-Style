# 关于 Zero-Style

**Zero-Style** 是一个面向设计师、开发者、品牌人的视觉风格图鉴,系统地收录和阐释 **52+ 种** 经典与现代的设计风格。

## 为什么做这件事

设计风格繁多,但分散在书籍、博客、博物馆里,缺乏一个统一的图鉴。
- 学习者:每学一种风格,都要从零搜集资料
- 设计师:在做品牌时,经常需要快速参考"类似风格"的代表作品
- 开发者:在实现 UI 时,需要了解风格背后的视觉逻辑,而不只是"看起来像"

Zero-Style 试图成为一本**可索引、可比较、可参考**的设计风格百科。

## 每种风格包含什么

- **风格描述** — 一段话讲清楚它是什么、来自哪里
- **核心特征** — 视觉与设计上的关键标志
- **配色方案** — 典型色板(含十六进制)
- **字体建议** — 配合的字体方向
- **适用场景** — 哪些项目适合用这种风格
- **代表案例** — 真实的品牌/作品
