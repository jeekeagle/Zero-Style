# 编辑设计(Editorial)

> 杂志的呼吸。

**起源年代**: 2010s · **分类**: 现代趋势

## 风格描述

源自杂志编辑设计的网页风格,以大字号衬线 + 强烈对比 + 精致排版 + Bento 网格为标志,常用于品牌故事、个人作品集。

## 核心特征

- 大字号
- 衬线体
- Bento 网格
- 强烈对比
- 故事感

## 字体建议

衬线 + 无衬线混排

## 适用场景

- 品牌故事
- 作品集
- 长文阅读

## 代表案例

- Bloomberg Businessweek
- Pitchfork

## 设计要点

设计 **编辑设计** 风格时,请记住:

1. **忠于核心特征** — 不要为了"看起来像 X"而堆砌装饰,而要让每个元素都为表达服务
2. **配色与字体协同** — 色彩与字体共同塑造风格情绪,任一改变都会削弱整体气质
3. **场景适配** — 即使是相同风格,在不同场景(网页/海报/包装)下的实现细节也不同
4. **时代背景** — 理解风格诞生的时代背景与文化土壤,避免无源之水的模仿
