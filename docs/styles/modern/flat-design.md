# 扁平化设计(Flat Design)

> 去除拟物。

**起源年代**: 2012 · **分类**: 现代趋势

## 风格描述

2012 年微软 Metro 与苹果 iOS 7 推动的扁平化设计,反对拟物化的渐变/阴影,使用纯色 + 简洁图形,塑造现代移动 UI 语言。

## 核心特征

- 纯色填充
- 无渐变
- 无阴影
- 极简图形
- 移动优先

现代无衬线(Roboto)

## 适用场景

- 移动 App
- 企业产品
- 图标设计

## 代表案例

- Windows 8 Metro
- iOS 7
- Google Material 1.0

## 设计要点

设计 **扁平化设计** 风格时,请记住:

1. **忠于核心特征** — 不要为了"看起来像 X"而堆砌装饰,而要让每个元素都为表达服务
2. **配色与字体协同** — 色彩与字体共同塑造风格情绪,任一改变都会削弱整体气质
3. **场景适配** — 即使是相同风格,在不同场景(网页/海报/包装)下的实现细节也不同
4. **时代背景** — 理解风格诞生的时代背景与文化土壤,避免无源之水的模仿
