# 日式禅意(Japanese Zen)

> 侘寂 wabi-sabi。

**起源年代**: 14世纪 · **分类**: 东方古典

## 风格描述

日本禅宗美学,以"侘寂"(wabi-sabi)为核心理念,接受不完美、无常、不对称,体现极简、自然、安静的东方哲学。

## 核心特征

- 极度留白
- 不对称(不对称美)
- 自然材质
- 单一红点/印章
- 静谧氛围

明朝体(细衬线)

## 适用场景

- 茶道/花道
- 日本料理
- 高端极简品牌

## 代表案例

- MUJI
- 无印良品
- 京都旅馆

## 设计要点

设计 **日式禅意** 风格时,请记住:

1. **忠于核心特征** — 不要为了"看起来像 X"而堆砌装饰,而要让每个元素都为表达服务
2. **配色与字体协同** — 色彩与字体共同塑造风格情绪,任一改变都会削弱整体气质
3. **场景适配** — 即使是相同风格,在不同场景(网页/海报/包装)下的实现细节也不同
4. **时代背景** — 理解风格诞生的时代背景与文化土壤,避免无源之水的模仿
