# 80年代复古(Retro 80s)

> 霓虹与磁带。

**起源年代**: 1980s · **分类**: 时代年代

## 风格描述

1980 年代是 MTV、合成器、Walkman 的时代,设计以霓虹渐变、网格透视、磁带/录像带元素、合成波字体为标志。

## 核心特征

- 霓虹渐变
- 网格透视
- 合成波字体
- 磁带元素
- 电子感

## 适用场景

- 音乐节
- 游戏
- 复古派对

## 代表案例

- Stranger Things 片头
- Drive 电影海报

## 设计要点

设计 **80年代复古** 风格时,请记住:

1. **忠于核心特征** — 不要为了"看起来像 X"而堆砌装饰,而要让每个元素都为表达服务
2. **配色与字体协同** — 色彩与字体共同塑造风格情绪,任一改变都会削弱整体气质
3. **场景适配** — 即使是相同风格,在不同场景(网页/海报/包装)下的实现细节也不同
4. **时代背景** — 理解风格诞生的时代背景与文化土壤,避免无源之水的模仿
