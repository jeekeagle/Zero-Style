# 像素复古(Pixel Retro)

> 低分辨率,高表达。

**起源年代**: 1980s · **分类**: 视觉表现

## 风格描述

8-bit/16-bit 时代游戏机的视觉遗产,以像素为单位绘制、限定调色板(8/16/64 色)、黑色硬边描边,塑造出鲜明的怀旧感。

## 核心特征

- 像素化图像
- 限定调色板(NES 56色等)
- 黑色硬边
- 等宽像素字体
- 方块构图

## 设计要点

设计 **像素复古** 风格时,请记住:

1. **忠于核心特征** — 不要为了"看起来像 X"而堆砌装饰,而要让每个元素都为表达服务
2. **配色与字体协同** — 色彩与字体共同塑造风格情绪,任一改变都会削弱整体气质
3. **场景适配** — 即使是相同风格,在不同场景(网页/海报/包装)下的实现细节也不同
4. **时代背景** — 理解风格诞生的时代背景与文化土壤,避免无源之水的模仿
