# 视觉表现(Visual Styles)

从极简到繁复的视觉语言

本分类收录 **10 种** 设计风格。

## 风格列表

- [极简主义 (Minimalism)](/styles/visual/minimalism) — *Less is more.*
- [孟菲斯 (Memphis)](/styles/visual/memphis) — *打破规则的派对。*
- [新拟物 (Neumorphism)](/styles/visual/neumorphism) — *柔软的浮雕。*
- [玻璃拟态 (Glassmorphism)](/styles/visual/glassmorphism) — *透明与模糊。*
- [赛博朋克 (Cyberpunk)](/styles/visual/cyberpunk) — *高科技,低生活。*
- [蒸汽波 (Vaporwave)](/styles/visual/vaporwave) — *消逝的消费主义。*
- [像素复古 (Pixel Retro)](/styles/visual/pixel-retro) — *低分辨率,高表达。*
- [瑞士国际 (Swiss Design)](/styles/visual/swiss-design) — *理性的网格。*
- [包豪斯 (Bauhaus)](/styles/visual/bauhaus) — *形式追随功能。*
- [装饰艺术 (Art Deco)](/styles/visual/art-deco) — *奢华的几何。*

## 分类说明

**视觉表现** 关注从极简到繁复的视觉语言。这些风格虽各有侧重,但都体现了人类对视觉表达的持续探索。
