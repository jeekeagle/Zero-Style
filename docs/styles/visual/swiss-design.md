# 瑞士国际(Swiss Design)

> 理性的网格。

**起源年代**: 1950s · **分类**: 视觉表现

## 风格描述

1950 年代瑞士发展出的平面设计风格,严格基于网格、Helvetica 字体的理性主义设计,强调客观性、可读性与功能性。

## 核心特征

- 严格网格
- Helvetica/Akzidenz-Grotesk
- 不对称布局
- 黑白主色 + 一红
- 照片裁切

## 适用场景

- 企业品牌
- 公共信息设计
- 博物馆导视

## 代表案例

- 瑞士航空
- Helvetica 纪录片
- Müller-Brockmann 海报

## 设计要点

设计 **瑞士国际** 风格时,请记住:

1. **忠于核心特征** — 不要为了"看起来像 X"而堆砌装饰,而要让每个元素都为表达服务
2. **配色与字体协同** — 色彩与字体共同塑造风格情绪,任一改变都会削弱整体气质
3. **场景适配** — 即使是相同风格,在不同场景(网页/海报/包装)下的实现细节也不同
4. **时代背景** — 理解风格诞生的时代背景与文化土壤,避免无源之水的模仿
