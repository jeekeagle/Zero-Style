# 包豪斯(Bauhaus)

> 形式追随功能。

**起源年代**: 1919 · **分类**: 视觉表现

## 风格描述

1919 年由 Walter Gropius 在德国创立的包豪斯学校,强调艺术与技术的统一,使用基本几何形状(圆/方/三角)和原色,深刻影响现代设计。

## 核心特征

- 基本几何形
- 三原色(红/蓝/黄)
- 无衬线字体
- 不对称构图
- 功能性至上

## 设计要点

设计 **包豪斯** 风格时,请记住:

1. **忠于核心特征** — 不要为了"看起来像 X"而堆砌装饰,而要让每个元素都为表达服务
2. **配色与字体协同** — 色彩与字体共同塑造风格情绪,任一改变都会削弱整体气质
3. **场景适配** — 即使是相同风格,在不同场景(网页/海报/包装)下的实现细节也不同
4. **时代背景** — 理解风格诞生的时代背景与文化土壤,避免无源之水的模仿
