# 配色方案(Color Schemes)

色彩的情感与象征

本分类收录 **7 种** 设计风格。

## 风格列表

- [莫兰迪 (Morandi)](/styles/color/morandi) — *灰调的温柔。*
- [马卡龙 (Macaron)](/styles/color/macaron) — *甜品的色谱。*
- [大地色 (Earth Tone)](/styles/color/earth-tone) — *泥土的颜色。*
- [霓虹荧光 (Neon)](/styles/color/neon) — *电光的色彩。*
- [单色调 (Monochrome)](/styles/color/monochrome) — *一色的深度。*
- [复古胶片 (Film Retro)](/styles/color/film-retro) — *胶片的颗粒。*
- [粉彩 (Pastel)](/styles/color/pastel) — *柔色的梦境。*

## 分类说明

**配色方案** 关注的情感与象征。这些风格虽各有侧重,但都体现了人类对视觉表达的持续探索。
